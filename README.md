### The 'No-Code' Advantage: A Core Design Principle

The absence of executable code in this repository is not a limitation; it is a deliberate architectural choice that provides five key advantages:

1.  **Universal Compatibility:** Platform-agnostic protocols.
2.  **Inherent Security:** Zero attack surface. You cannot hack a metaphor.
3.  **Radical Transparency:** Human-readable "source code".
4.  **Future-Proof Stability:** Built on language, not libraries that rot.
5.  **Reproducibility:** Guaranteed by a manual calibration procedure.
